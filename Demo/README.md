# 💰 Smart Money — AI Financial Assistant

**Full-stack AI-powered financial management system**

Built with React + Flask + MongoDB + ML (scikit-learn/numpy/pandas)

## Quick Start

```bash
# 1. Backend
cd backend && pip install -r requirements.txt
python seed_data.py   # seed sample data
python app.py         # starts on port 5000

# 2. Frontend (new terminal)
cd frontend && npm install && npm start  # starts on port 3000
```

**Login:** `srusti@example.com` / `password123`

See `docs/README.md` for full documentation.

## Features
- ✅ JWT Authentication
- ✅ Income & Expense Tracking (full CRUD)
- ✅ AI Auto-Categorization (NLP keyword engine)
- ✅ Budget Planning & Alerts
- ✅ Financial Goals Tracker
- ✅ Spending Insights Engine
- ✅ Expense Prediction (Linear Regression)
- ✅ Anomaly / Overspending Detection (Z-Score)
- ✅ Financial Health Score (0-100)
- ✅ Future Savings Simulation (3 scenarios)
- ✅ Investment Suggestions (risk-based)
- ✅ Subscription / Recurring Detection
- ✅ Middle-Class Budget Mode
- ✅ Interactive Charts (Recharts)
- ✅ Dark Mode UI with responsive design
